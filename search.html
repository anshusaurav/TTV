[game-id] Search by game id
{gorgc} Search by streamer username
viewer:1000 Streams with more than 1000 viewers
lang:en Streams in English langauge

game/user[ viewer[ lang]]

Need to use any of game or user in above format


if user can be entered as {gorgc} where gorgc is username

otherwise [29595] where 29595 is gameid


Other three arguments are optional and their order is flexible.

[29595] lang:en viewer:1000

[29595] viewer:1000 lang:en

Returns the same result
